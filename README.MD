# 🎋 国风取名神器 · Chinese Name Generator

> 一款融合「国风书法美学」与「命理五行算法」的宝宝取名神器  
> 采用 React + Ant Design Mobile + Node.js + Express 构建  
> 支持生肖分析、五行补益、音律评分、风格选取，一键生成寓意佳名

---

## ✨ 功能亮点

### 🧠 智能命理算法
- 支持输入宝宝出生日期，自动分析八字五行  
- 提示所需五行与生肖喜忌  
- 自动生成符合命理平衡的高分名字  

### 🌸 国风雅致 UI
- 书法卷轴风格界面  
- 宣纸色背景 + 金色渐变按钮  
- 名字卡片带淡入上浮动画（fadeInUp）  
- 仿大师测名报告式详情页，含寓意解析 / 五行分析 / 音律评分  

### ⚙️ 灵活配置
- 前端使用 **Vite + React + Antd Mobile**
- 后端使用 **Express + TypeScript + dayjs**
- 可轻松接入数据库（如 Supabase / MongoDB）或 AI 接口扩展  

### 💳 可拓展功能
- 接入微信 / 支付宝 / Stripe 支付接口，实现付费解锁报告  
- 接入 AI 接口（Gemini / GPT / Claude）生成诗词出处与高级命理分析
